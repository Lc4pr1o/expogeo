AgroExport v1.0.8

Plugin QGIS para exportação de linhas de orientação (guidance lines) para terminais agrícolas John Deere e PTx Trimble.

Instalar via QGIS

No QGIS: Plugins → Gerenciar e Instalar Plugins → Configurações → Adicionar

Nome: AgroExport
URL:  https://lc4pr1o.github.io/expogeo/plugins.xml

Após adicionar o repositório, pesquise por AgroExport na aba Todos e clique em Instalar.

Formatos exportados

  • John Deere GEN4 — .zip para Operations Center
  • John Deere GS3_2630 — SpatialCatalog para cartão SD
  • PTx Trimble AgData — .agf para Precision-IQ (GFX-750/1050/1060)
  • PTx Trimble AgGPS — Shapefile para terminais legado
