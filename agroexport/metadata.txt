changelog=
    1.0.6 - Importacao como ponto de entrada do fluxo
        * Aba Importacao movida para posicao 1
        * Abas Padronizacao, Classificacao e Exportacao ficam bloqueadas ate importar
        * Apos importar, plugin avanca automaticamente para Padronizacao
        * Se camadas ja existem no projeto, todas as abas sao liberadas normalmente
    1.0.5 - Divisao em blocos, importacao de shapefile e melhorias de interface
        * Campo Talhao na classificacao vira selector de nomenclatura (Parte/Bloco/Gleba)
        * Divisao automatica em blocos quando projeto excede 2.5 MB
        * Preview de blocos com reorganizacao de talhoes via drag-and-drop
        * Nova aba Importacao: carrega shapefile .shp diretamente no projeto
        * Indicador de tamanho estimado do projeto na aba Padronizacao
        * Janela de conclusao de exportacao simplificada (sem caminhos individuais)
    1.0.4 - Exportacao direta em pasta, sem compactacao
        * Arquivos exportados diretamente nas pastas Gen4/, GS3_2630/, AgData/, AgGPS/
        * Sem mais arquivos .zip individuais por formato
        * Mensagem "Exportando... aguarde." aparece durante o processo
    1.0.3 - Preservacao de curvas e controle de qualidade
        * Remoção de vértices próximos só ocorre se desvio < tolerância geométrica
        * Vértices em curvas são sempre preservados independente do espaçamento mínimo
        * Campo "Tolerância geométrica" controla o quão agressiva é a simplificação
        * Densificação por espaçamento máximo mantida (não altera traçado)
    1.0.2 - Algoritmo de padronização de vértices corrigido
        * Substitui Douglas-Peucker por regularização real de espaçamento
        * Mínimo/Máximo agora controlam distância em metros entre vértices consecutivos
        * Vértices muito próximos (< mín) são removidos sem distorcer o traçado
        * Gaps maiores que o máximo recebem vértices interpolados linearmente
        * A linha resultante nunca se afasta da geometria original
        * Aba renomeada para "Padronização"
    1.0.1 - Melhorias de usabilidade e desempenho
        * Aba Exportação: agrupamento por marca (John Deere / PTX Trimble)
        * Aba Exportação: remove prefixo e metadados de fallback
        * Aba Classificação: carregamento de feições até 10x mais rápido
        * Remove GS3 legacy XML (.isg)
    1.0.0 - Versão inicial
        * Exportação para John Deere OC (Gen4)
        * Exportação para GS3-2630
        * Exportação para PTx Trimble Precision-IQ (AgData)
        * Exportação para PTx Trimble AgGPS legado
        * Simplificação Douglas-Peucker
        * Classificação por cliente/fazenda/talhão
